# 3. Controls 

1 Player Mode:

Use up arrow and down arrow keys to move the paddle up and down.

2 Player Mode:

Player 1: Use up arrow and down arrow keys to move the paddle up and down.  
Player 2: Use w key to move up and s key to move down.
